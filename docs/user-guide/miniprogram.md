# MiniMall Mini-Program Guide

## Getting Started

1. Open WeChat and search "MiniMall"
2. Launch the mini-program
3. Login with WeChat (automatic)

## Browse Products

- Home: featured products, categories
- Category tab: browse by category
- Search: tap icon, enter keywords

## View Product

- Tap card for details
- Select specs if applicable
- Stock status shown

## Shopping Cart

**Add:** Product page > "Add to Cart" or quick add from card

**Manage:**
- +/- buttons for quantity
- Delete to remove
- "Clear Cart" for all

## Checkout

1. Go to Cart
2. Tap "Checkout"
3. Select/add address
4. Add remark (optional)
5. Tap "Place Order"

## Payment

1. Tap "Pay"
2. WeChat Pay dialog
3. Authenticate
4. Notification on complete

## My Orders

**Statuses:** Pending, Paid, Shipped, Delivered, Cancelled

**Actions:**
- Cancel: pending only
- Confirm receipt: delivered
- View tracking: shipped orders

## Addresses

- Add: tap "+" > fill > save
- Edit: tap address > modify
- Delete: swipe left
- Default: tap "Set as Default"

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Cannot add to cart | Login required |
| Payment fails | Check WeChat balance |
| Order not showing | Pull down to refresh |
| Image not loading | Check connection |
