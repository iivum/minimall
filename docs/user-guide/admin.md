# Admin Dashboard Guide

## Login

1. Open admin portal
2. Enter credentials + WeChat verification

## Dashboard

View: orders today, revenue today, total products, pending orders

## Products

**Add:** Products > Add New > fill details > Publish

**Edit:** Products > All Products > Edit > Update

**Stock alerts:** Items below 10 highlighted in red

## Orders

**Filters:** Status, date range, customer search

**Process:** Click order > verify payment > update to Shipped > add tracking > notify

**Export:** Click Export for CSV (ID, customer, amount, status, date)

## Categories

- Two-level: Parent > Child
- Drag-and-drop reorder
- Delete only if no products assigned

## Statistics

Period: Today, Week, Month, Year

Metrics: orders count, revenue, avg order value, conversion

Charts: order trend, revenue bar, category pie

## Settings

- Store info: name, contact, hours
- Admin account: change password, update profile
- Payment: WeChat merchant credentials + certificate

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Login fails | Check WeChat permission |
| Payment fails | Verify merchant ID and certificate |
| Image upload fails | Check file size (<2MB) |
| Orders not updating | Clear cache and retry |
