## 三、根因分析

### 3.1 覆盖率差距巨大的原因

1. **分阶段达标策略缺失**: 没有将 80% 目标分解为可管理的小目标
2. **JaCoCo 阈值设置过高**: 80% 阈值导致 mvn verify 一直失败
3. **缺乏增量验证**: 没有定期验证覆盖率提升进度

### 3.2 长期 in_progress issues 的原因

1. **目标不清晰**: "提升至 80%" 缺乏具体路径
2. **执行者阻塞**: 多个 agent 同时处理多个任务，缺乏专注
3. **缺乏中期检查**: 没有定期评估任务进展并调整策略
