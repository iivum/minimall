## ADR 模板

```markdown
# ADR-XXXX: 标题

## 状态
[已通过 | 已驳回 | 已废弃 | 审议中]

## 背景
描述做出该决策的问题上下文。

## 决策
具体说明所做的决定。

## 后果
- 正面影响
- 负面影响
- 需要关注的问题

## 决策日期
YYYY-MM-DD

## 决策者
[人员名称]
```

## 使用指南

1. **新增决策**: 创建新 ADR 文件，遵循模板格式
2. **索引更新**: 新增 ADR 后更新本文件索引
3. **状态变更**: 当决策状态变更时，更新对应 ADR 状态
