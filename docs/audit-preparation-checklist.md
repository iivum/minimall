**操作指引**:
1. 登录微信公众平台 → 开发管理 → 开发设置
2. 获取 AppID（格式：wxxxxxxxxxxxxxxxxx）
3. 更新 `miniprogram/app.js` 中的 `APP_ID` 常量
4. 提交代码变更

**当前占位符值**:
```javascript
const APP_ID = 'YOUR_APPID'  // 需要替换为真实 AppID
```

**目标值示例**:
```javascript
const APP_ID = 'wx1234567890abcdef'  // 替换为真实 AppID
```

### 2.3 生产域名备案确认

| 检查项 | 状态 | 确认人 | 备注 |
|--------|------|--------|------|
| 已有已备案的生产域名 | ☐ 待确认 | - | - |
| 域名已配置 SSL 证书 (HTTPS) | ☐ 待确认 | - | - |
| 域名已在微信公众平台白名单 | ☐ 待确认 | - | - |
| 域名解析生效 | ☐ 待确认 | - | - |

**操作指引**:
1. 确认域名已在阿里云/腾讯云等服务商备案
2. 申请 SSL 证书并配置 HTTPS
3. 在微信公众平台 → 开发管理 → 服务器域名 中添加 request 合法域名
4. 验证域名可访问：`curl -I https://your-domain.com`

**需要配置的域名白名单**:
```
request 合法域名：https://api.minimall.com  (示例)
```

### 2.6 服务器域名白名单配置

| 检查项 | 状态 | 确认人 | 备注 |
|--------|------|--------|------|
| request 合法域名已配置 | ☐ 待确认 | - | - |
| uploadFile 合法域名已配置（如有文件上传） | ☐ 待确认 | - | - |
| downloadFile 合法域名已配置（如有文件下载） | ☐ 待确认 | - | - |
| 域名配置已同步到代码 | ☐ 待确认 | - | - |

**当前 app.js 配置**:
```javascript
const BASE_URL = 'http://localhost:8080/api'  // 需要替换为生产域名
```

**目标配置**:
```javascript
const BASE_URL = 'https://api.minimall.com/api'  // 替换为已备案的生产域名
```

**操作指引**:
1. 在微信公众平台 → 开发管理 → 服务器域名 中添加以下域名：
   - `request 合法域名`: api.minimall.com
   - `downloadFile 合法域名`: static.minimall.com（如有）
2. 更新 `app.js` 中的 `BASE_URL` 配置
3. 提交代码变更
4. 确认小程序可以正常调用生产环境 API
