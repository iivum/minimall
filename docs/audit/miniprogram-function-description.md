# 微信小程序功能说明文档

## 基本信息

| 项目 | 内容 |
|------|------|

## 核心功能模块

### 1. 用户模块
- 微信一键登录（自动创建账户）
- 查看个人资料
- 查看会员等级和权益
- 积分余额查询

### 2. 商品模块
- 首页推荐商品展示
- 分类浏览商品
- 商品搜索功能
- 商品详情查看（价格、库存、描述）

### 3. 购物车模块
- 添加商品到购物车
- 修改商品数量
- 删除购物车商品
- 清空购物车

### 4. 订单模块
- 创建订单（下单）
- 订单支付（微信支付）
- 查看订单列表（待支付/已支付/已完成/已取消）
- 取消订单（待支付状态）
- 确认收货（已发货状态）

### 5. 支付模块
- 微信支付集成
- 支付成功/失败提示
- 支付回调处理

### 6. 会员权益模块
- 会员等级查询（普通/银卡/金卡/钻石）
- 会员专属优惠提示
- 积分规则说明

## 用户交互流程

### 首页浏览流程
1. 用户打开小程序 → 自动微信登录
2. 首页展示推荐商品和分类入口
3. 用户可切换「分类」Tab 浏览
4. 点击商品卡片进入商品详情

### 购买商品流程
1. 商品详情页选择规格（如有）→ 加入购物车
2. 购物车页面确认商品 → 点击结算
3. 确认订单信息（地址、备注）→ 提交订单
4. 调起微信支付 → 支付成功后跳转订单页

### 订单管理流程
1. 「我的」页面 → 「我的订单」
2. 查看各状态订单列表
3. 待支付订单 → 可取消或去支付
4. 已发货订单 → 可确认收货

## 数据收集说明

### 必须收集的信息
| 信息类型 | 用途 | 收集方式 |
|----------|------|----------|
| 微信 OpenID | 用户身份识别 | 微信登录时自动获取 |
| 昵称/头像 | 展示用户信息 | 微信授权获取 |
| 手机号 | 订单联系 | 用户主动填写（非强制） |

### 操作信息
| 信息类型 | 用途 |
|----------|------|
| 浏览记录 | 推荐商品 |
| 订单数据 | 订单管理、售后 |

| 信息类型 | 用途 | 授权时机 |
|----------|------|----------|
| 收货地址 | 订单配送 | 下单时申请 |
