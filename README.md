# Minimall

MVP微信商城系统 - 面向个体户的微信小程序电商解决方案

## 项目结构

```
minimall/
├── backend/          # Spring Boot 后端服务
├── miniprogram/      # 微信小程序前端
├── admin-ui/         # 管理后台前端 (Vue 3 + TypeScript)
├── docs/             # 项目文档
│   ├── sprints/      # 迭代回顾文档
│   └── meetings/     # 会议记录
└── SPEC.md           # 规格说明书
```

## 技术栈

### 后端
- Spring Boot 3.2.0 / Java 17
- PostgreSQL 15
- MyBatis-Plus 3.5.5

### 前端
- 微信小程序 (原生开发)
- Vue 3 + TypeScript + Vite (管理后台)
